Privacy Policy

Autodarts Session Coach

This extension does not collect, transmit, or store personal data externally.

Data Processing

The extension processes visible content from the Autodarts website in order to provide real-time performance analysis.

Local Storage

Any data used by the extension is stored locally in the user's browser and is not transmitted to external servers.

Third Parties

No user data is shared with third parties.

Remote Code

This extension does not use remotely hosted code.
